About The Website

This is a cryptocurrency website that retrieves 100 coins from an API and displays them on the DOM. When the user clicks the "More Info" button, it fetches the price details of a specific coin from another API. When the user checks a coin it adds the coin the live report chart and display the usd price in the chart(updates every 2 seconds). The user can add only 5 coins to live report chart, when 6 coin checked a modal pops up asking the user if he want to change a coin or cancel and not add the coin..
